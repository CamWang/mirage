# 1
## 题目描述
输入$a$和$b$，输出$a+b$的结果。
## 输入格式
一行两个正整数$a$和$b$。
## 输出格式  
一行一个正整数$a+b$。
## 样例
输入
```
12
```
输出
```
3
```
## 数据范围
对于$100%$的数据，$1<=a$, $b<=10^6$
